# google-apps-script
Google Apps Script Projects


## API-Rest-Functions

Este projecto crea una API REST que recibe un body mediante una petición POST con la estructura:

```json
{
	"action": "capitalize",
	"value": "HOLaaa mundO"
}
```

Y retorna la ejecución de la función:

```json
{
	"status": 200,
	"result": "Holaaa Mundo"
}
```
